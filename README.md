# Sales Data Analysis using Excel

## Objective
To analyze sales data and identify trends, top products, and profitability insights.

## Dataset
Sample Superstore dataset.

## Tools Used
- Excel / WPS Spreadsheet

## Analysis Performed
- Data cleaning (removed duplicates, fixed date formats)
- Monthly sales trend analysis using pivot tables and line chart
- Top 5 products analysis by sales
- Profit analysis by category

## Key Insights
- Sales peak during Q4, indicating seasonal demand
- A small number of products contribute a large share of total sales
- Certain categories are more profitable than others

## Conclusion
This project demonstrates the use of Excel for real-world business data analysis and decision-making.
